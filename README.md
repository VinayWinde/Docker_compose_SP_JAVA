# 🚀 Spring Boot Java Application with Docker Compose

This project demonstrates how to containerize a Spring Boot application and run it using Docker Compose.

---

## 🧱 Technologies Used

- Java 17+
- Spring Boot
- Maven / Gradle
- Docker
- Docker Compose
- PostgreSQL / MySQL (optional)

---
